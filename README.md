# CRU LAB Complete Website

完整 Next.js 15 專案，包含：

- 首頁
- 關於 CRU LAB
- AI 配方設計
- 原料資料庫
- 研究中心
- 品牌建立
- CRU SHOP

## 啟動

```bash
npm install
npm run dev
```

然後開啟 `http://localhost:3000`。

## 建置

```bash
npm run build
npm start
```

圖片已放在 `public/images/`，所有頁面皆為響應式設計。

## Image update
This package includes the latest CRU LAB luxury visual assets in `public/images/`.
